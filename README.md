# hermpy - A Python package for Mercury space science

[![docs](https://github.com/daraghhollman/hermpy/actions/workflows/docs.yml/badge.svg)](https://hermpy.readthedocs.io/en/stable/)

Tools for aiding in the handling of data, their analysis, and the creation of publication ready plots from spacecraft around Mercury.

See our [documentation](https://hermpy.readthedocs.io/en/stable/) for more details.
